Для запуска проекта для начала нужно установить зависимости

Для этого в директории в консоли нужно прописать 

pip install -r requirements.txt

Далее можно запустить проект командой

uvicorn main:app --reload    

Где по адресу локалхоста http://127.0.0.1:8000/docs#/default/token_input_token_input_post
запустится сервис

В пост-запрос /login/get_token для получения корректного ответа вводим
логин и пароль одного id из database_security из файла base.py

Система выдаст нам валидный токен, который будет активен в течение 30 минут
Далее во второй пост-запрос /token/unput вводим данный токен
и получаем данные о зарплате и дате повышения id пользователя, под которым 
мы заходили в систему
